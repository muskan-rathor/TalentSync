# TalentSync — Microservices Job Portal

A scalable job portal built with microservices architecture.

## Tech Stack
- **Backend:** Node.js, TypeScript, Express.js
- **Message Broker:** Apache Kafka
- **Database:** MongoDB, Redis
- **AI Integration:** Gemini AI
- **Auth:** JWT Authentication

## Services
- **Auth Service** — Register, Login, JWT
- **Job Service** — Job listings, applications
- **Users Service** — User profiles
- **Utils Service** — Kafka consumer, Email notifications
